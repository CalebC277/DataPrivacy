## Installation and Setup

### Setup

- Python 3.6+
- These are the required libraries
  - folium
  - geopy
  - requests
  - numpy
  - pandas
  - matplotlib
  - tqdm

Install all of these packages:

```bash
pip install folium geopy requests numpy pandas matplotlib tqdm
```

### Build and Compile

All of these files 
  - hybrid.py
  

can be built and comiled by:

```bash
python3 file_name.py
```
