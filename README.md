## Installation and Setup

### Setup

- Python 3.6+
- These are the required libraries
  - folium
  - geopy
  - requests
  - numpy
  - pandas
  - matplotlib
  - tqdm

Install all of these packages:

```bash
pip install folium geopy requests numpy pandas matplotlib tqdm
```

### Build and Compile

All of these files can be built and comiled by:

```bash
python3 hybrid.py
```
```bash
python3 poi.py
```
```bash
python3 walkable.py
```
```bash
python3 compare.py
```
